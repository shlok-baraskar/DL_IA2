Problem Statement
Accumulation of dust, snow, bird drops, and physical/electrical damage on solar panels reduces their efficiency. This project builds a deep learning classifier to automatically detect and classify solar panel surface conditions into 6 categories:

Class	Description
Clean	Normal, functioning solar panel
Dusty	Dust accumulated on surface
Bird-drop	Bird droppings on panel
Electrical-damage	Electrical faults visible
Physical-Damage	Cracks or physical damage
Snow-Covered	Panel covered with snow
Approach: Transfer Learning using EfficientNetB0 pre-trained on ImageNet, fine-tuned on solar panel images.
